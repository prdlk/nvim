# Releases

## bun-v1.3.1: Bun v1.3.1

**Published**: 2025-10-22

To install Bun v1.3.1
```bash
curl -fsSL https://bun.sh/install | bash
# or you can use npm
# npm install -g bun
```

Windows:
```bash
powershell -c "irm bun.sh/install.ps1|iex"
```

To upgrade to Bun v1.3.1:
```bash
bun upgrade
```
